# Bean Chillin’ – Unity Game Jam Project

## Description
A single-player Unity game developed for **Mini Jam** within a **3-day game jam**.  
The project focuses on **rapid prototyping, player movement, obstacle interaction, and gameplay feedback systems** under strict time constraints.

This game was built to practice **scoped feature development, fast iteration, and core gameplay implementation** in Unity.

---

## Features
- Player movement and physics-based control  
- Obstacle and pickup interaction systems  
- Gameplay-driven visual and audio feedback  
- Simple UI and game state flow  
- Modular C# scripts designed for rapid iteration  

---

## Technologies
- Unity  
- C#
